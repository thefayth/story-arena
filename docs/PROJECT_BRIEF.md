# Project Brief

## What This Is

Story Arena is a private-first AI screenwriting platform and public title-board concept. It lets writers develop and protect work privately, generate public-safe metadata, receive AI feedback, prepare IP-support materials, and choose what appears publicly.

## Why It Exists

Many creator platforms reward public disclosure before protection. Story Arena reverses that pattern. It treats the private vault as the center and public visibility as an approval-controlled output.

## Product Shape

- Private writer vault
- AI feedback panel
- Script-genome summaries
- IP prep center
- Approved-only public title pages
- Monthly awards board
- Canva-led visual system
- WordPress public runtime

## Canonical Runtime

The deployable website remains the private WordPress plugin on FaithCheltenham.com. This repository is a public surface, not the runtime.

## Public Promise

Scripts stay private unless the writer opts in to public release.

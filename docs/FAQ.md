# FAQ

## Is this the source code?

No. This is a protected public project surface, not the full source code or runtime system.

## Are screenplays public here?

No. Private scripts, uploads, drafts, and raw AI context are excluded.

## Can people submit scripts through this repository?

No. The repository is informational. Submissions belong in the private Story Arena flow on the live site when enabled.

## Can someone reuse the brand or workflow?

No. All rights are reserved.

## Why use GitHub if the source is private?

The repository establishes ownership, legitimacy, public documentation, workflow visuals, and a controlled project surface without releasing the engine.

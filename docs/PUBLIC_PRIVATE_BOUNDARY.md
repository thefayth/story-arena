# Public / Private Boundary

## Public

- Project overview
- Approved summaries
- Public title metadata
- Public rules
- Awards surface
- Sanitized script-genome concepts
- Brand and workflow visuals
- WordPress page copy
- Public-safe gallery assets

## Private

- Full screenplay text
- Raw uploads
- Private drafts
- Owner IDs and account data
- Attachment IDs and storage keys
- Prompt context and private AI instructions
- Credentials and deployment systems
- Admin routes and operational code
- Private legal, medical, benefits, family, and business records

## Rule

Private creative work may influence private analysis. Public visuals and public metadata must come only from approved, sanitized briefs.

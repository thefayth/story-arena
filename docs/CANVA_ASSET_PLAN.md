# Canva Asset Plan

Canva remains the master creative kit for polished exports. This public package includes a strong starter asset set, but production Canva exports should replace raster drafts as they become available.

## Hero Image

- Purpose: README and WordPress hero
- Dimensions: 1600 x 900
- Style direction: golden pop-cinema stage, Black women screenwriters visibly leading a diverse creative ensemble, script cards, private vault cues
- Color direction: champagne, ivory, gold, cinema black, rose, teal
- Typography direction: bold editorial title with database label system
- Canva prompt: Create a polished golden pop-cinema hero image for Story Arena, a private-first AI screenwriting vault and public title board. Center Black women screenwriters as directors, builders, and creators within a diverse ensemble. Show screenplay title cards, sealed vault motifs, warm stage lighting, and creator-owned public approval energy. No celebrity likenesses, no copied film frames, no raw script text.
- Alternate prompt: Design a cinematic website hero for a protected screenwriter awards system. Bright champagne and gold palette, database title wall, private vault iconography, inclusive creator ensemble, modern editorial layout.
- Alt text: Golden Story Arena hero with diverse screenwriters, title cards, and a private screenplay vault.

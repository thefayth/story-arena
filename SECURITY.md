# Security

## Reporting

Report suspected security, privacy, or ownership issues through FaithCheltenham.com contact channels.

Do not open public issues containing credentials, private drafts, server paths, tokens, nonces, account details, or private screenshots.

## Scope

This public repository is not the production codebase. The private WordPress runtime, upload storage, AI processing flow, and deployment systems are intentionally excluded.

## Security Principles

- Private scripts stay private unless explicitly approved for public release.
- Public pages use approved metadata and sanitized script-genome summaries.
- Raw script text, owner IDs, attachment IDs, prompt context, and private file storage details should not appear in public payloads.
- The public repository must not contain credentials, operational logs, private workflows, or deployment secrets.
